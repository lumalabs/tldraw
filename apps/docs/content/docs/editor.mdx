## Code Blocks

### Inline Code

### Block Code

```python
# Python example
def greet(name):
    return f"Hello, {name}!"
print(greet("Alice"))
```

```js
// JavaScript example
function greet(name) {
	return `Hello, ${name}!`
}
console.log(greet('Alice'))
```
